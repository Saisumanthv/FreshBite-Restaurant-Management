# FreshBite Restaurant Management

Welcome to **FreshBite**, a modern, full-stack restaurant management solution designed to streamline restaurant operations, order tracking, and menu management.

## 🚀 Tech Stack

This project is built using modern web technologies:

- **Language:** TypeScript  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Database & Backend:** Supabase (with migrations support)  

## 📌 Features

- **Menu Management:** Easily update dishes, pricing, and availability.  
- **Order Tracking:** Real-time monitoring of customer orders.  
- **Dashboard:** An intuitive interface for restaurant staff.  
- **Responsive Design:** Fully optimized for mobile and desktop screens.
